# data-analytics-projects

## SalesAnalysis
This project analyzes retail sales data to identify trends in revenue,
top performing product categories, and regional performance.

Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn

##AI Assisted Sales Analysis
This project analyzes retail sales data to identify trends in revenue,
top performing product categories, and regional performance. The AI-generated insights are currently simulated to demonstrate the workflow. The pipeline is fully ready to integrate with a real LLM API.

Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn
